# Installing Trial Versions

1) Add the Bryntum NPM registry
npm config set "@bryntum:registry=https://npm.bryntum.com"

2) Install one or more needed components
npm install @bryntum/grid@npm:@bryntum/grid-trial@{version}
npm install @bryntum/scheduler@npm:@bryntum/scheduler-trial@{version}
npm install @bryntum/schedulerpro@npm:@bryntum/schedulerpro-trial@{version}
npm install @bryntum/gantt@npm:@bryntum/gantt-trial@{version}
npm install @bryntum/taskboard@npm:@bryntum/taskboard-trial@{version}
